# About Project

Chat from socket

## Требования

* PHP >= 8.1
* Composer >= 2
* make >= 4


## Install and start project
Установка докера и docker-compose

curl -sSL get.docker.com | sh
sudo apt-get install docker-compose

make setup - запуск вебсокета
make start-back - запуск сервера

* `make lint`
* `make test`
